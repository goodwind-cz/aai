## Dispatch attention items — independent adjudication

   is probed (file-system candidates only). Candidate ORDER is sound against
   user-writable shadowing: admin-writable `Program Files` paths come FIRST,
   PATH-derived entries LAST (aai-run-tests.ps1:90-115), so a user-dir

   inert, and the suite passes AAI_UNAME per-invocation (no export leak).
   Activation on macOS/Linux would require an explicitly exported
   `AAI_UNAME=MSYS…`, which is not an accident shape. Namespacing further
   (e.g. AAI_TEST_UNAME) is optional polish, not required; the override is
   documented in the header as test-only.
